В настоящее время существует две наиболее популярные IDE для web-разработки: 

  • Visual Studio Code: https://code.visualstudio.com/
  • WebStorm: https://www.jetbrains.com/ru-ru/webstorm/ (платная)

Каждая из них представляет достаточно большой набор интрументов для решения ежедневных задач. Если кратко - то WebStorm обладает большими возможностями "из коробки", а Visual Code - требует установки и настройки плагинов.
