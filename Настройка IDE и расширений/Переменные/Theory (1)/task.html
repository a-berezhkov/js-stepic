Объявление переменных

Для объявления переменных можно использовать следующие конструкции:

let a = 5;

const b = 6;

При этом значения 5 и 6 записываются в переменные a и b соответственно.

Такая операция (=) называется операцией присваивания. 

Разница в let и const заключается в том, что:

  • с помощью let объявляется переменная, которую можно изменить;
  •  const объявляет константу, которая не поддается изменениям.

К примеру, код ниже выполнится отлично.

let a = 10;

a = 2;

А вот с константой так уже не получится:

const a = 10;

a = 2; // Ошибка!

Очень удобно иногда читать код буквально. К примеру let означает "пусть" а const - константа. Тогда все обретает смысл - "пусть a будет равно 5"!

Важно отметить, что константы должны быть инициализированы.

const a; //  Uncaught SyntaxError: Missing initializer in const declaration
const a = 5;

В переменной или константе можно хранить любые данные: число, строку, массив и т.д.

let stringWithText = "Любой текст, который вам нравится";

let stringWithNumber= 123488;

Можно записать объявление переменных и в одну строчку, но так код становится менее читабельным. 

let stringWithText = "Любой текст, который вам нравится", stringWithNumber= 123488

Почему это так важно? Дело в том, что любой продукт - это тысячи строк кода, а создает его команда разработчиков. Очень тяжело понять, как работает чужой код (а иногда и свой), поэтому очень важно делать его понятным для всех.

Еще можно копировать данные из одной переменной в другую.

let someVar = 111
let newVar = someVar

Хотя можно и не задавать значение переменной при ее объявлении.

let someVar = 111
let newVar;
newVar = someVar;

 А вот чего нельзя - это два раза объявлять переменную с одним именем.

let a = 10;
let a = 5; // Ошибка
