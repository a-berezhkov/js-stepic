Немного о функциях

Подробно функции описаны в соответствующем разделе, но так как периодично мы будем их видеть до того как пройдем, было бы неплохо представлять что это такое.  

Даже если что-то из этого пока будет непонятно, ничего страшного, смело листайте дальше!

Что нам нужно знать.

Несколько строчек кода можно объединить в функцию, которую потом можно "вызвать", то есть выполнить код, который внутри. Делается, в самом простом случае это с помощью ключевого слово function и названия функции, которое вы придумываете самостоятельно. Для вызова функции пишется ее имя и скобочки ()

//Функция
function ItsFunction(){
    console.log("Hello");
    console.log("I'm function");
}

//Вызов функции
ItsFunction() // Hello 
              // I'm function

//Можно вызываит сколько угодно раз
ItsFunction() // Hello 
              // I'm function

Еще одна важная деталь, которая будет полезна для понимания — параметры. Это то, что функция может принимать. К примеру, имя пользователя

//Функция
function ItsFunction(name){
    console.log("Hello");
    console.log("I'm", name);
}

//Вызов функции
ItsFunction("Ivan") // Hello 
              // I'm Ivan

//Можно вызываит сколько угодно раз
ItsFunction("Oleg") // Hello 
              // I'm Oleg

Еще важно, что в определённом контексте функции могут назвать "методами".

Пока этого будет достаточно. 

Есть много стандартных функций в JavaScript. К примеру метод alert(), который рассмотрим далее.
