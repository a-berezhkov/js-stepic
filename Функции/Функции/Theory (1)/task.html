Параметры функции

Ранее уже задавались параметры функции и все отлично работало. Но что будет, если функция требует параметр, а он не передан?

function showMessage(anySimpleArray) {
    for (let name of anySimpleArray) {
        console.log(`Привет, ${name} !`)
    }
}

showMessage()

Такой код выдаст ошибку. 

Что же делать, если существует возможно того, что придет пустой массив или в некоторых случаях использование параметра не обязательно? Разработчики JS подумали об этом и сделали возможным установить значение по-умолчанию для параметров функции.

Для этого в параметрах функции необходимо присвоить значение по-умолчанию. При этом вызов без параметра не вызовет ошибку.  Для этого всего лишь необходимо присвоить параметру функции значение при ее объявлении.

const message = ["Игорь", "Илья", "Петр", "Иван", "Кирилл"]

function showMessage(anySimpleArray = []) {
    for (let name of anySimpleArray) {
        console.log(`Привет, ${name} !`)
    }
}

showMessage(); // ничего не выведет

showMessage(message); 

//     Привет, Игорь !
//     Привет, Илья !
//     Привет, Петр !
//     Привет, Иван !
//     Привет, Кирилл !

В функцию можно передать любое количество параметров 

const message = ["Игорь", "Илья", "Петр", "Иван", "Кирилл"];

const obj = {item: "value"};

const str = "Строка";

const int = 123;

function showMessage(message, obj, str, int) {
    console.log(message, obj, str, int);
}

showMessage(message, obj, str, int); //[ 'Игорь', 'Илья', 'Петр', 'Иван', 'Кирилл' ] { item: 'value' } Строка 123

И даже если неизвестно, сколько будет параметров, можно использовать специальный оператор ...  в этом случае полчится массив внутри функции

const message = ["Игорь", "Илья", "Петр", "Иван", "Кирилл"];

const obj = {item: "value"};

const str = "Строка";

const int = 123;

function showMessage(...ourArr) {
    console.log(ourArr);
}

showMessage(message, obj, str, int);

// [
//     [ 'Игорь', 'Илья', 'Петр', 'Иван', 'Кирилл' ],
//         { item: 'value' },
//         'Строка',
//         123
// ]
