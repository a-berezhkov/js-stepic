Булевый тип (boolean)

Булевый тип мы будем использовать постоянно, хотя он и имеет только два варианта значения - true и false. Можно провести аналогию с выключателем — либо он подает ток, и лампочка горит (true) либо выключена (false).

Обычно он используется в различных условиях или при необходимости хранить какое-то состояние. Иногда булевые переменные называют "флагом". 

let isSale = true; //является ли товар акционным
let isAgree = false; // дано ли согласие на обработку персональных данных

Иногда вместо true и false используется единица и ноль, поэтому JavasScript в любой непонятной ситуации будет пытаться привести их в число или строку.

true+true+true; //3
true+"string"; //'truestring'
