parseInt и parseFloat

Преобразовать строку в число, если конечно в ней число, можно с помощью функции Number() или проще с помощью +.

let num = "5" ; тип - строка
Number(num); тип - число

let num2 = "6" ; тип - строка
+num; тип - число

И числами это работает предсказуемо. Но что будет если мы подадим строку, содержащее число?

Number("55abc"); // NaN
+"55abc"; // NaN

Предсказуемо. Но все-таки очень хочется получить только цифры, так как пользователь при вводе могу просто ошибиться. 

Как раз тут и помогут наши функции:

parseInt("55.66abc"); // 55
parseFloat("55.66abc"); // 55.66

Но если в начале будет текст, а потом цифры, то мы получим NaN, так как функции последовательно считывают цифры до первого символа

parseInt("abc55.66"); // NaN
parseFloat("abc55.66"); // NaN

Пустая строка интерпретируется как 0 во всех числовых функциях
