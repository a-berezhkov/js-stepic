isFinite() и isNaN()

isNaN()

А давайте проверим, является ли число NaN равным числу NaN. Для такой проверки используется оператор сравнения ==.

NaN==NaN; //false

Неожиданно. Но вы наверное уже не удивляетесь. Действительно, NaN не равен NaN. 

Окей, давайте использовать метод isNaN для проверки

isNaN(NaN); //true
isNaN(100); //false

Вроде логично. А давайте проверим строку 

isNaN("str"); //true

Удивительно! Но в данном случае есть хотя бы объяснение. isNaN() сначала пытается преобразовать значение в число и лишь потом проверяет, равно ли оно NaN. Надо напомнить, что преобразование строки в число дает NaN, если эту операцию совершить невозможно. Поэтому пример ниже работает вполне нормально.

isNaN("100"); //false

isFinite() 

isFinite()  проверяет, является ли число числом. Проще всего понять принцип работы на примере:

isFinite("15"); // true
isFinite("str"); // false, потому что специальное значение: NaN
isFinite(Infinity); // false, потому что специальное значение: Infinity
