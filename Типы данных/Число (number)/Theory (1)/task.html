Математические операции

С числами можно делать все арифметические операции:

  • Сложение +,
  • Вычитание -,
  • Умножение *,
  • Деление /,
  • Взятие остатка от деления %,
  • Возведение в степень **. 

Давайте приведем несколько примеров каждой операции.

Сложение и вычитание

let num = 12;
let num2 =3;
len result = num + num2; //15

result = result + num2; // 18

result  = 5 + 5 + 5; //15

result = 100 - 1; // 99

Прежде чем перейти дальше, уточним несколько терминов, которые можно часто услышать.

Операнд – то, к чему применяется оператор. 

5 + 100

В коде выше есть два операнда: левый операнд равен 5, а правый операнд равен 100.

Унарным называется оператор, который применяется к одному операнду. Унарный минус "-" меняет знак числа на противоположный.

let num = 10;
num = -num

Это еще не вся магия!

Можно сократить запись сложения или вычитания с помощью операций (+=) и (-=).

let num = 12;
let num2 = 3;
len result = num + num2; //15

result += num2; // 18

result  += 5; // 23

result -= 100; // -77

Фокус в том, что запись result += num2аналогична result = result + num2.

И еще один фокус, который используется при написании скриптов - операции инкремента и декремента. Все просто: так как увеличение или уменьшение на 1 в программировании используется регулярно, то ее сократили до i++ и i-- что равносильно i = i +1 и i = i-1.

let i = 0;
i++; // 1
i++; // 2
i++; // 3
i--; //2

Умножение  и деление

let num = 12;
let num2 =3;
len result = num * num2; //36

result = result * num2; // 

result  = 5/5; //1

result = 100 / 1;  // 100

Эту запись так же можно упростить, аналогично сложению и вычитанию.

result *= num2;
result /= num2;

Остаток и степень 

С возведением в степень все более - менее понятно.

let two = 2;
two = two ** 2;// 4
two = two ** 2;// 16

Но и тут есть одно интересное свойство - можно вычислить корень или даже кубический корень возведя в степени 1/2 и 1/3

4 ** (1/2) // 2

С остатком чуть посложнее. Это удобный инструмент решения многих задач, к примеру, определение четных и нечетных чисел. 

let num = 6;
let i = num % 3; // 0
let j = num % 5; // 1

И в заключении - Таблица приоритетов. Да, да, да.... Так же как и в математике все операции имеют свой приоритет выполнения. Особенно это интересно сказывается на операции присваивания.

let a = 1;
let b = 2;

let c = 3 - (a = b + 1);

a ; // 3
c ; // 0
