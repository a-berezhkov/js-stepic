Оператор typeof

Оператор typeof возвращает строку, указывающую тип операнда.

console.log(typeof 965); // "number"

console.log(typeof 'Привет!'); // "string"

console.log(typeof true); // "boolean"

console.log(typeof var); //"undefined"

typeof возвращает следующие значения в зависимости от типов (обрати внимание на null)

Тип Результат
Undefined "undefined"
Null "object" 
Boolean "boolean"
Number "number"
String "string"
Symbol (новый тип из ECMAScript 2015) "symbol"
Function object "function"
Любой другой тип "object"
