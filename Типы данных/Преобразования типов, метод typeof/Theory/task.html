Преобразования типов

Преобразование типов может потребоваться если необходимо явно преобразовать строку в число или наоборот. В зависимости от ситуации нам может быть удобно работать с переменной как строкой  или как с числом.

Строки

К примеру, если мы хотим сложить разряды числа 123, то удобней представить его в виде строки "123", так как тогда мы получим доступ к каждому символу. 

let str = String(123);
console.log(typeof str); //string

Числа

Преобразование в число часто происходит неявно, при использовании математических операций (если это возможно) 

console.log("10"/"2"); //5

Но это можно сделать и явно

let str = String(123);
console.log(typeof str); //string
let num = Number(str)
console.log(typeof num); //number

Для чисел справедливо следующие преобразование 

численного преобразования:

Значение Преобразуется в…
undefined NaN
null 0
true / false 1 / 0
string Пробельные символы (пробелы, знаки табуляции \t, знаки новой строки \n и т. п.) по краям обрезаются. Далее, если остаётся пустая строка, то получаем 0, иначе из непустой строки «считывается» число. При ошибке результат NaN.

Булевые (bool) 

Все, что нужно знать о преобразовании:

  • Значения 0, пустая строка, null, undefined и NaN, становятся false.
  • Все остальные значения становятся true
Boolean(123);// true
Boolean(0);// false
Boolean("0");// true
Boolean("string");// true
Boolean(NaN); //false
Boolean(null); //false
