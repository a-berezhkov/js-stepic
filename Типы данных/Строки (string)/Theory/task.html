Типы кавычек

Главное в этом разделе - не запутаться в кавычках, их будет много.

Для объявления строки используются любые кавычки - "" , '' или ``(для ` - нажимаете на букву ё с клавишей shift)

let s1 = "Строка раз"
let s2 = 'Строка два'
let s3 = `Строка три`

Возникает логичное желание разобраться в таком количестве кавычек. 

Между '' одинарными и  "" двойными кавычками нет разницы.

А вот в обратные кавычки `` позволяют вставлять внутрь вычисляемые выражения с помощью конструкции ${…}

let name = "Егор";
`Привет, ${name}!`; // Привет, Егор!

В обратные кавычки может вставлять более одной строки

let fruitList = `
 - Apple
 - Banana
 - Cherry
`;

Чаще для переноса строк используется специальный символ - \n 

let fruitList = "- Apple\n- Banana\n- Cherry";

Есть и другие спецсимволы: 

\', \" - Кавычки (если необходимо их добавить внутри строки). При этом сам \ часто называют "символом экранирования". 

\uXXXX  - Символ в кодировке UTF-16 с шестнадцатеричным кодом XXXX, например, \u00A9 — Юникодное представление знака копирайта, ©.

\\ - Служит для отображения слеша внутри строки.

Остальные можно найти с помощью поисковой системы.

Прежде чем продолжить работу со строками, потребуется научиться выводить и получать данные для удобной отладки.
