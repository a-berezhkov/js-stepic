Конкатенация

Осталась одна важная вещь. Было бы классно соединяться строки между собой. Для этих целей можно использовать оператор + или метод str.сoncat() объединяет текст из двух или более строк и возвращает новую строку. Обратите внимание, что метод сoncat() можно вызвать только у объекта типа String

Рассмотри на примере

let a = "Я";
let b= "строка";

console.log(a +" " + "новая" + " " + b); // Я новая строка
console.log("".concat(a," ","новая", " ", b)); // Я новая строка
