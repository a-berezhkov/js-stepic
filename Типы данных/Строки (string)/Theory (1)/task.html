Доступ к символам 

Еще одной особенностью строки является то, что мы можем получить доступ к ее символам. Проще показать это  на примрере

let str = `Я строка`;

//Превая буква
console.log( str[0] ); // Я
console.log( str.charAt(0) ); // Я

//Третья буква
console.log( str[2] ); // с
console.log( str.charAt(2) ); // с

//Последняя буква
console.log( str[str.length - 1]) //  а

Наверное, первый вопрос — почему при [0] получается первый элемент? Ответ довольно простой — почти вся нумерация строк и массивов начинается с 0. На то есть исторические причины и привычка программистов. Вообще цифра в скобочках — это индекс элемента.

Можно представить, что каждая буква лежит в своей ячейке

[]/индекс 0 1 2 3 4 5 6 7
Символ Я   с т р о к а

Аналогично работает и метод CharAt() в который мы передаем индекс. 

Возможно мы захотим получить элемент, которого нет в исходной строке.

let str = `Я строка`;
console.log(str[999]); // undefined

 ChatAt() в этом же случае вернет пустую строку. 

let str = `Я строка`;
str.charAt(999); // ""

А давайте попробуем заменить букву "Я" на "я"

let str = 'Я строка';

str[0] = 'я'; // ошибка

 Не получилось. А жаль. Но зато можно сказать, что строки неизменяемый тип данных.
