О типах данных

На собеседованиях любят спрашивать, сколько типов данных вы знаете. Отвечайте смело - восемь. 

Вот они:

  1. Undefined (Неопределённый тип) 
  2. Boolean (Булев, Логический тип)
  3. Number (Число)
  4. String (Строка)
  5. BigInt (Ну оооочень большое число)
  6. Null ("ничего")
  7. Symbol ("уникальный идентификатор")
  8.  Object (Объект) 

Все они простые, кроме Object. Два типа мы пока пропустим: Symbol и BigInt - рассмотрим их только в дополнительных материалах. Так как до конца курса прекрасно можно обойтись и без них.

Так же Object будет рассмотрен в отдельном разделе, про него можно написать почти целую книгу. Зато тут рассмотрим метод typeofи преобразование типов.
