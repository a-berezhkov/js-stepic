Логические операторы

Существует четыре логических оператора - || (ИЛИ), && (И) и ! (НЕ) и ?? (Оператор нулевого слияния).

Рассмотри сразу на простом примере.  Нам нужна программа, которая проверяет пароль пользователя на длину - не менее 6 символов и не более 8. 

Для этого можно написать вот такой код

(6<password) && (password<10)

Только в случае, если password будет от 6 до 10 символов выражение примет значение true

true && true // true

Для того, что бы понимать какой результат будет, нужно запомнить таблицу истинности

A B || (ИЛИ) && (И)
true true true true
false false false false
false true true false
true false true false

Не (!) используется как отрицание

!true; // false
!false; // true

Приоритет НЕ ! является наивысшим из всех логических операторов, поэтому он всегда выполняется первым, перед && или ||

Приоритет оператора && больше, чем у ||

Оператор нулевого слияния (??)

Результат выражения a ?? b будет следующим:

  1. если a определено ( не является null или undefined) , то a,
  2. если a не определено ( является null или undefined), то b.
a = 1;
b = 1;

a ?? b; // 1

a = null;
b = 1;

a ?? b; // 1

a = undefined;
b = 1;

a ?? b; // 1

a = 1;
b = null;

a ?? b; // 1

a = 1;
b = undefined;

a ?? b; // 1

a = undefined;
b = undefined;

a ?? b; // undefined

a = null;
b = undefined;

a ?? b; // undefined

a = undefined;
b = null;

a ?? b; // null

Приоритет оператора ?? такой же, как и у ||.
