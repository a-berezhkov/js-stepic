Строки (часть 2)

На самомо деле в JS все типы данных представлены в виде объектов. Это длинная история и мы оставим ее на потом. Сейчас важнее другое  - как и в реальном мире, у объектов есть различные свойства. Так же с объектами мы можем совершать какие-то действия (методы).

Предположим у нас есть стол, он обладает следующими свойствами:

  • высотой;
  • шириной;
  • длиной;
  • цветом;

И мы можем выполнить с ним следующие дейтсвия:

  • перенести;
  • выкинуть;
  • изменить цвет.

В JS что бы получить дотсуп к свойству или методу  объекта используется символ ..

К примеру если хотим получить длину стола то достаточно написать: стол.длина. А если этой свойство доступно для изменения - то можем и задать его стол.длина = 100. 

Для методов все аналогично, только сам метод еще необходимо вызвать - стол.выкинуть().

Как уже писалось ранне, почти у свех объектов есть свои свойства и методы.

Свойство length

Свойство length - позволяет получить длину строки

console.log( `My`.length ); //2

Доступ к символам 

Еще одной особенностью строки является то, что мы можем получить доступ к ее символам. Проще показать это  на примрере

let str = `Я строка`;

//Превая буква
console.log( str[0] ); // Я
console.log( str.charAt(0) ); // Я

//Третья буква
console.log( str[2] ); // с
console.log( str.charAt(2) ); // с

//Последняя буква
console.log( str[str.length - 1]) //  а

Наверное первый вопрос - почему при [0] получается первый элемнет? Ответ довольно простой - почти вся нумерация строк и массивов начинается с 0. На то есть исторические причины и привычка программистов. Вообще цифра в скобочках - это индекс элемента.

Можно представить что каждая буква лежит в своей ячеке

[]/индекс 0 1 2 3 4 5 6 7
Символ Я   с т р о к а

Аналогично работает и метод CharAt() в который мы передаем индекс. 

Возможно мы захотим получить элемент, которого нет в исходной строке.

let str = `Я строка`;
console.log(str[999]); // undefined

 ChatAt() в этом же случае вернет пустую строку. 

let str = `Я строка`;
console.log(CharAt[999]); // ""

А двайте попробуем заменить букву "Я" на "я"

let str = 'Я строка';

str[0] = 'я'; // ошибка

 Не получилось. А жаль. Но зато можно сказать, что строки неизменяемый тип данных.

Осталась одна важная вещь. Было бы классно соединяться строки между собой. Для этих целей можно использовать оператор + или метод str.сoncat() объединяет текст из двух или более строк и возвращает новую строку. Обратите внимание, что медот сoncat() можно вызвать только у объекта типа String

Рассмотри на примере

let a = "Я";
let b= "строка";

console.log(a +" " + "новая" + " " + b); // Я новая строка
console.log("".concat(a," ","новая", " ", b)); // Я новая строка
