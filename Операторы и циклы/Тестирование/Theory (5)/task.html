Стрелочные функции

Тестировщики выступают против стрелочных функций, так как они имеют свой контекст выполнения. Поэтому конструкция ниже на сработает (пример взят с документации), так как this имеет контекст функции.

describe('my suite', () => {
  it('my test', () => {
    // should set the timeout of this test to 1000 ms; instead will fail
    this.timeout(1000);
    assert.ok(true);
  });
});
