HOOKS

Hooks (хуки) - это события которые будут автоматические вызываться при запуске теста.

Всего есть четыре хука - before(), after(), beforeEach() и afterEach().

before() - запуститься один раз перед первым тестом в блоке

after() - запуститься один раз после последнего теста в блоке

beforeEach() - запускается перед каждым тестом в  блоке

afterEach() - запускается после каждого теста в блоке

Пример кода хуков 

describe('hooks', function () {
  before(function () {
    // запуститься один раз перед первым тестом в этом блоке
  });

  after(function () {
    // запуститься один раз после последнего теста в этом блоке
  });

  beforeEach(function () {
    // запускается перед каждым тестом в этом блоке
  });

  afterEach(function () {
    // запускается после каждого теста в этом блоке
  });

  //  тесты
});

Тесты могут появляться до, после или вперемешку с вашими хуками. Хуки будут запускаться в том порядке, в котором они определены; все before()хуки запускаются (один раз), затем любые beforeEach()хуки, тесты, любые afterEach()хуки и, наконец, after()хуки (один раз). 

Любой хук можно вызвать с необязательным описанием, что упрощает выявление ошибок в ваших тестах. Если хуку присвоена именованная функция, это имя будет использоваться, если не указано описание.

beforeEach(function () {
  // beforeEach hook
});

beforeEach(function namedFun() {
  // beforeEach:namedFun
});

beforeEach('Хук beforeEach запущен', function () {
  // beforeEach:Хук beforeEach запущен
});
